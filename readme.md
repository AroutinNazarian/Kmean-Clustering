# K-Means Clustering Implementation

Machine learning implementation of K-Means clustering algorithm using cosine similarity and Euclidean distance metrics.

---

## Overview

From-scratch implementation of K-Means clustering with multiple distance metrics. Demonstrates core concepts of unsupervised learning including centroid initialization, iterative assignment, and convergence.

---

## Algorithms Implemented

**K-Means Clustering:**
- Random centroid initialization
- Iterative clustering with distance-based assignment
- Centroid recalculation and convergence
- Multiple stopping criteria

**Distance Metrics:**
- **Euclidean Distance:** L2 norm for feature space distance
- **Cosine Similarity:** Angular distance for high-dimensional data

---

## Key Features

✓ Complete K-Means implementation from scratch  
✓ Support for both Euclidean and cosine distance metrics  
✓ Convergence analysis and iteration tracking  
✓ Interactive Jupyter notebook with visualizations  

---

## Technologies

**Python Libraries:** pandas, numpy, matplotlib  
**Concepts:** Unsupervised learning, clustering, distance metrics, centroid-based algorithms
